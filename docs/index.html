核心功能

硬件匹配检测

自动检测 CPU、内存、硬盘等真实配置，识别参数造假

电池健康评估

检测电池健康度、循环次数，评估电池剩余寿命

硬盘状态检测

读取 SMART 状态、通电时长，评估硬盘健康程度

交互式功能检测

屏幕坏点、键盘按键、触控板、摄像头等一站式检测

翻新检测

检测官方翻新标记、第三方部件更换、企业退役设备

PDF 报告导出

生成专业检测报告，方便保存和分享

  • macOS 10.13 (High Sierra) 或更高版本
  • 仅支持 Apple Silicon (M1/M2/M3)
  • 约 50 MB 磁盘空间

Windows

  • Windows 10 (1809) 或更高版本
  • 支持 x64 架构
  • 约 50 MB 磁盘空间
